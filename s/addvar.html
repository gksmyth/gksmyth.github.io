add.var Added variable plots for current linear model
DESCRIPTION
Produces an added variable plot for each covariate in a linear model.
 
USAGE
add.var(lm.obj,...)
 
REQUIRED ARGUMENTS
lm.obj - a linear model object
 
OPTIONAL ARGUMENTS
Any argument suitable for the function drop1
 
EXAMPLES
lm.obj <- lm(y ~ x1+x2)
par(mfrow=c(1,2))
add.var(lm.obj) # produces both added variable plots
