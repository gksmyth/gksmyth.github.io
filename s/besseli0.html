bessel.i0   Modified Bessel function
DESCRIPTION
Returns the modified Bessel function of order zero, I0(x), for numeric-valued vectors and arrays x.
 
USAGE
bessel.io(x)
 
DETAILS
Attributes of x are preserved in the returned values. NA's in x produce NA's in the returned values.
 
REFERENCE
Abramowitz, M., and Stegun, I. A. (1965). Handbook of Mathematical Functions, Dover, New York.
 
EXAMPLES
x <- c(0.1,3.2,4.5,6.7)
bessel.i0(x) # modified Bessel function of components of x
